# unity 오브젝트 객체 추적
---

## 설명
`MoveTowards` 함수를 이용한 일정 거리만큼 객체를 추적하는 기능구현. <br>
지정된 목표 위치로 객체를 부드럽게 이동시키는 방식. <br>
`Abs` 를 사용한 거리별 추적 속도조절. <br>

## 기능
- 목표 지점으로 객체를 일정 속도로 이동
- 지정된 거리만큼 추적 후 정지
- 거리비례 추적속도 변동
- 거리별 색깔 변환

## 버전
- Unity 6000.0.43f1 (universal 3D)
- C# .NET Standard 2.1

## 사용 방법
1. manager안에 코드 삽입. 수치값 조절(코드 내에서 변경해도 반영 안될 수도 있음)
2. obj안에 모든 오브젝트 있음.
3. 카메라는 탑뷰.
